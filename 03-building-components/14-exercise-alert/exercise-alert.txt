Exericse: Showing an Alert
--------------------------
- Based on the previous exercise's <Button /> component
    - Render a single <Button /> component in <App />

    - When the component is clicked
        - It renders a dismissable <Alert /> component
            - Wher it is rendered with a close button
            - When clicked the component is removed from the DOM
            - Reference (https://getbootstrap.com/docs/5.3/components/alerts/#dismissing)

        - A "child" of string data type is passed to the <Alert /> component
            - The content is "My Alert"
